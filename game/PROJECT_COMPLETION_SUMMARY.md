## 🎯 项目概况

### 核心理念
一款基于认知心理学的互动叙事生存游戏，玩家扮演团队领导者，在极端环境下做出理性决策，识破15种认知偏误，带领团队在零下30度的暴风雪中生存10天。

### 教育价值
- **寓教于乐**: 将抽象心理学概念转化为游戏体验
- **批判性思维训练**: 通过实战学习识别认知偏误
- **企业决策模拟**: CEO视角下的真实管理困境

## 💡 经验总结

### 技术经验
1. **TypeScript是必需的** - 避免运行时错误
2. **Zustand简化状态管理** - 轻量级且强大
3. **本地化要尽早考虑** - 后期重构成本高
4. **测试驱动开发** - 提高代码质量
5. **构建优化很重要** - 影响用户体验

### 游戏设计经验
1. **教育内容要自然融入** - 不要生硬说教
2. **选择要有真实代价** - 增加沉浸感
3. **平衡需要大量测试** - 数据驱动决策
4. **角色差异化要明显** - 增加重玩价值
5. **叙事与机制结合** - 提升体验

### 项目管理经验
1. **文档要详细完整** - 方便协作和维护
2. **阶段性交付** - 控制范围和风险
3. **用户反馈很珍贵** - 及时调整方向
4. **自动化很重要** - 减少重复劳动
5. **持续优化** - 没有完美的第一版
