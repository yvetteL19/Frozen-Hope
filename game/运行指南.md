# 🚀 运行指南

## 方式一：命令行运行（推荐）

### 1. 打开终端

在Mac上按 `Command + Space`，输入 `Terminal` 并打开

### 2. 进入项目目录

```bash
cd "/Users/yiwei/Desktop/Frozen Hope/game"
```

### 3. 安装依赖（首次运行）

```bash
npm install
```

等待安装完成（可能需要几分钟）

### 4. 启动开发服务器

```bash
npm run dev
```

### 5. 打开浏览器

在终端会显示类似以下信息：
```
  VITE v5.0.8  ready in 500 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
```

在浏览器中打开 `http://localhost:5173`

### 6. 开始游戏！

选择角色，开始你的冰封求生之旅。

## 常见问题

### Q: 遇到"npm: command not found"

**A**: 需要先安装Node.js

1. 访问 https://nodejs.org/
2. 下载并安装LTS版本
3. 安装完成后重启终端

### Q: 端口被占用（Port 5173 is in use）

**A**: 更改端口或关闭占用的程序

方式1 - 使用其他端口：
```bash
npm run dev -- --port 3000
```

方式2 - 查找并关闭占用5173端口的程序

### Q: 样式没有加载

**A**: 确保Tailwind CSS正确安装

```bash
npm install -D tailwindcss postcss autoprefixer
```

### Q: 如何清除游戏存档？

**A**: 在浏览器控制台（F12）运行：

```javascript
localStorage.removeItem('frozen_hope_save')
```

然后刷新页面。
