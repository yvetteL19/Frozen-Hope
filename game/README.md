# 冰封希望：最终幸存者

一款基于认知心理学的互动叙事游戏。在极端环境下，你能保持理性，识破认知偏误，带领团队存活10天吗？

## 🎮 游戏特色

- **6个独特角色**：CEO、程序员、助理、向导、飞行员、销售总监
- **15种认知偏误**：权威偏误、沉没成本、幸存者偏差等
- **14个核心事件**：第一幕5个、第二幕4个、第三幕5个
- **4个结局**：救援、惨胜、崩溃、你的终点
- **深度复盘系统**：详细的认知偏误分析和现实案例

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发服务器

```bash
npm run dev
```

然后在浏览器中打开 `http://localhost:5173`

### 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist` 目录中。

## 📖 游戏说明

### 核心机制

1. **压力时钟**：团队的集体理智值，达到12格游戏结束
2. **玩家HP**：你的生命值，降至0游戏结束
3. **信标修复进度**：达到100%可获得最佳结局
4. **NPC状态**：冷静、激动、恐慌，影响他们的建议

### 角色选择建议

- **新手推荐**：助理或程序员（难度⭐⭐）
- **中等难度**：向导或销售总监（难度⭐⭐⭐）
- **困难模式**：CEO或飞行员（难度⭐⭐⭐⭐）

### 游戏流程

```
开始 → 选择角色 → 第一幕(第2-3天) → 第二幕(第4-7天) → 第三幕(第8-10天) → 结局 → 心理复盘
```

## 🎯 完美决策提示

- 识破认知偏误，选择理性选项可获得**-1压力时钟**
- 合理使用角色技能（但注意代价）
- 注意NPC的心理状态（恐慌时的建议往往是陷阱）
- 优先照顾伤者和病人
- 团队合作比个人英雄主义更重要

## 🏆 成就系统

- **完美理性者**：单局游戏中没有陷入任何认知陷阱
- **团队守护者**：所有NPC存活到结局
- **完美救援**：达成结局A且所有NPC存活
- **最快救援**：在第5天前达成结局A

## 🔄 游戏存档

游戏会自动保存进度到浏览器的LocalStorage。刷新页面后可以继续之前的进度。

要清除存档，可以在浏览器控制台运行：
```javascript
localStorage.removeItem('frozen_hope_save')
```

## 🎨 自定义与扩展

### 添加新事件

1. 在 `src/data/events-actX.ts` 中添加事件定义
2. 定义事件的前置条件、选项和后果
3. 添加角色专属选项（可选）

### 添加新角色

1. 在 `src/data/roles.ts` 中添加角色定义
2. 在事件中添加该角色的专属选项

### 添加新认知偏误

1. 在 `src/data/biases.ts` 中添加偏误信息
2. 在事件中引用该偏误类型
