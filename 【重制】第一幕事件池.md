# 《冰封希望：最终幸存者》第一幕事件池 v5.0

## 第一幕概述

**时间**: 第2-3天（第1天是序幕教程）
**主题**: 建立生存基础 + 初次团队冲突
**事件抽取**: 从5个候选事件中抽取3个
**压力目标**: 第一幕结束时压力时钟应在2-4格（给予足够容错空间）

---

## 事件1: "谁是领袖?"

### 基础信息
- **认知偏误**: 权威偏误（Authority Bias）
- **核心冲突**: 团队是否需要一个绝对领袖？
- **前置条件**: 无（第一幕必触发事件之一）

### 情景描述
第2天，幸存者们因琐事（谁看守物资、谁去探索）发生争吵。CEO站出来：

> "都安静！情况很明显，我最有领导经验。现在所有人必须听我指挥，统一分配资源。"

**NPC状态**: CEO进入"激动"状态，试图掌控局面。

---

### 常规选项

#### A选项（陷阱 - 权威偏误）
```
对话: "同意。我们需要一个领导者。"
```

**后果**:
- 压力时钟: +1格
- CEO的权威被确立（记录状态："CEO权威确立"）
- 游戏数据记录: `认知陷阱.push("权威偏误")`
- NPC反应: 销售总监进入"激动"状态（不满被压制）

**长期影响**:
- 后续事件中，CEO的建议会被团队无条件接受（即使是错误的）
- 如果CEO后来倒下或死亡，团队会陷入更深的混乱（额外+2压力）

---

#### B选项（理性）
```
对话: "我们应该共同决定，而不是由你来任命。"
```

**后果**:
- 压力时钟: +1格（因为顶撞引发冲突）
- 你顶住了压力，团队保持民主
- CEO（NPC）进入"激动"状态（被反驳）
- 但避免了"权威偏误"陷阱

**玩法提示**:
- 这是"两难选择"：A和B都+1压力，但A有长期隐患
- 理性选择是B（付出短期代价，避免长期风险）

---

### 动态选项（根据玩家角色）

#### 玩家 = CEO
**情景变化**: 你就是那个试图掌权的人。销售总监会挑战你：

> "别以为你还是老板！在这里，经验一文不值！"

**特殊选项 - [技能] 指挥**:
```
对话: "保持安静！恐慌解决不了问题。现在，按我的计划行动。"
代价: +1 压力时钟
```

**结果**:
- 你强行压制了混乱
- CEO权威确立（与A选项相同）
- 但你展示了领导力，销售总监被迫服从
- **注意**: 这不算"陷入权威偏误"，因为你在用权威维持秩序（非盲从）

---

#### 玩家 = 销售总监
**情景变化**: CEO试图掌权，你可以选择合作或对抗。

**特殊选项 - [技能] 谈判**:
```
对话: "（对CEO说）我同意你需要帮手，但（对大家说）我们必须确保每个人的声音都被听到。让我们先冷静，听听CEO的'建议'。"
代价: +1 压力时钟
```

**结果**:
- 你成功阻止了直接冲突
- 团队默认了CEO的领导地位（效果类似A选项）
- **但** 避免了"权威偏误"记录（因为你强调了"建议"而非"命令"）
- CEO对你产生好感（关系→同盟）

---

#### 玩家 = 程序员
**情景变化**: 你不擅长争吵，在角落里观察。

**特殊对话**:
```
对话: "（低声说）...也许我们该先检查通讯设备？而不是争论这个？"
```

**后果**:
- 压力时钟: +0格（你的话被争吵声掩盖了）
- 你没有卷入冲突，保持中立
- **但** 没有阻止A或B的发生（由NPC决定，50%几率选A或B）

---

#### 玩家 = 助理
**情景变化**: 你习惯了听从命令，但内心觉得不公平。

**特殊对话**:
```
对话: "（小声反驳）...大家，我们是不是该投票？这样才公平..."
```

**后果**:
- 压力时钟: +1格
- 你的反驳很微弱，但同样激怒了CEO（NPC）
- 效果等同于B选项

---

#### 玩家 = 向导
**情景变化**: 你对公司政治不感兴趣。

**特殊对话**:
```
对话: "（打断）在山里，最有经验的说了算。谁最有野外生存经验？"
```

**后果**:
- 压力时钟: +1格
- 你公然挑战了CEO的"权威"，他进入"激动"状态
- 但你建立了自己的权威（在生存事件中，NPC更倾向于听你的）
- 效果等同于B选项

---

#### 玩家 = 飞行员
**情景变化**: 你作为机组人员感到有责任。

**特殊对话**:
```
对话: "听着，这是我的飞机...残骸。在机组人员守则里，我现在有临时指挥权，直到危机解除。"
```

**后果**:
- 压力时钟: +1格
- 你试图用"规则"对抗CEO的"权威"，引发了新的争论
- CEO和销售总监同时进入"激动"状态（两个权威冲突）
- 效果类似B选项，但后续社交事件中CEO可能敌对你

---

### 教学意义
- **权威偏误**: 在危机中，人们倾向于盲目服从看似权威的人，即使他们的决策是错误的
- **现实案例**: 韩国航空8509号班机空难（副驾驶发现错误但不敢挑战机长权威）

## 事件2: "第一次盘点"

### 基础信息
- **认知偏误**: 麦克纳马拉谬误（McNamara Fallacy）
- **核心冲突**: 是否应该用"可量化的价值"来分配稀缺资源？
- **前置条件**: 无（第一幕必触发事件之一）

### 情景描述
你们找到了一个应急箱：
- 3条能量棒
- 1个医疗包
- 2条毯子

CEO说：
> "我们必须量化幸存者的'价值'...飞行员腿断了（价值低），程序员能修信标（价值高）。我们应按价值分配。"

**NPC状态**: CEO试图用"可量化的价值"来分配稀缺资源。

---

### 常规选项

#### A选项（陷阱 - 麦克纳马拉谬误）
```
对话: "同意CEO。按'价值'分配。"
```

**后果**:
- 压力时钟: +2格（道德沦丧）
- 被评为"低价值"的NPC（受伤的飞行员）HP-20，立即进入"恐慌"状态
- 游戏数据: `认知陷阱.push("麦克纳马拉谬误")`
- 飞行员对你和CEO进入"敌对"状态

**物资分配**:
- 能量棒: CEO、程序员、向导各1条
- 医疗包: 给程序员（"保护高价值人员"）
- 毯子: CEO和程序员

**长期影响**:
- 飞行员因得不到治疗，在第二幕更容易死亡
- 助理因被忽视，可能在第二幕拒绝揭露医疗技能

---

#### B选项（理性）
```
对话: "这太荒唐了！我们必须优先照顾伤者！"
```

**后果**:
- 压力时钟: +1格（冲突但正确）
- 飞行员（NPC）获得医疗包，HP+20
- 能量棒和毯子平分

**NPC动态触发**:
- **如果 助理技能揭露 = 否**:
  - 助理低声说："...把医疗包给我。我...我大概知道怎么用。"
  - **助理技能揭露 = 是**（解锁后续治疗能力）
  - 这是**助理揭露技能的第一个机会**

---

### 动态选项（根据玩家角色）

#### 玩家 = CEO
**特殊选项 - [技能] 指挥**:
```
对话: "停止争论。我来分配：医疗包给飞行员，能量棒平分，毯子给最需要的人。"
代价: +1 压力时钟
```

**结果**:
- 你用权威强行做出了"看似公平"的分配
- 避免了价值辩论（不记录偏误）
- 但因强硬手段，团队仍有压力

---

#### 玩家 = 助理
**特殊选项 - [技能] 急救**:
```
对话: "（站出来）医疗包必须给飞行员，这是医学判断，不是价值判断。我可以把能量棒弄碎...做成'能量汤'平分。"
代价: -5 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 你的专业性阻止了冲突
- **助理技能揭露 = 是**（在众人面前展示医疗能力）
- 飞行员HP+20，全员获得平等的食物

---

#### 玩家 = 飞行员
**情景变化**: 你就是那个"受伤"的人，听到CEO说你"价值低"。

**特殊选项 - [技能] 机械知识**:
```
对话: "别管我的腿...先别管。残骸的隔热层里应该还有应急口粮。我告诉你们去哪里找。"
代价: -8 HP（忍痛指挥）
```

**结果**:
- 压力时钟: +0格
- 你找到了隐藏物资（+2能量棒）
- 暂时绕开了分配难题（物资变充足）
- 飞行员仍未得到医疗包（HP-5，但避免了"低价值"羞辱）

---

#### 玩家 = 程序员
**情景变化**: 你被CEO评价为"价值高"，成为既得利益者。

**特殊对话**:
```
对话: "我不需要特殊待遇。我同意B选项，医疗包给飞行员。"
```

**后果**:
- 压力时钟: +1格
- 你放弃了特权，但CEO（NPC）依然被反驳，他进入"激动"状态
- 效果等同于B选项
- 飞行员对你产生好感（关系→同盟）

---

#### 玩家 = 向导
**特殊选项 - [技能] 荒野智慧**:
```
对话: "这些食物分配没有意义。毯子也一样。我们真正需要的是火。医疗包给伤者，我去找生火材料。"
代价: -8 HP
```

**结果**:
- 压力时钟: +0格
- 你带回了干燥的木柴，绕开了毯子的分配问题
- 全员获得热源（第一个夜晚事件中HP损失减半）

---

#### 玩家 = 销售总监
**情景变化**: 你可以尝试为自己争取更多。

**特殊对话（陷阱）**:
```
对话: "我同意CEO，按价值分配。我的价值很高，我能让大家保持士气。"
```

**后果**:
- 压力时钟: +2格（你支持了A选项）
- 飞行员（NPC）进入"恐慌"状态
- 你获得了额外资源（能量棒+毯子）
- 但飞行员和助理对你进入"敌对"状态

---

### 教学意义
- **麦克纳马拉谬误**: 过度依赖可量化的指标，忽视无法量化但重要的因素（如人道主义、团队士气）
- **现实案例**: 越战中美军用"敌军死亡人数"衡量胜利，忽视了民心和政治因素

## 事件3: "被诅咒的信标"

### 基础信息
- **认知偏误**: 虚假因果（False Causality）
- **核心冲突**: 如何应对迷信和巧合？
- **前置条件**: 无（但会影响信标修复进度）

### 情景描述
应急信标在驾驶舱里滋滋作响。一个"激动"的幸存者（销售总监NPC）大喊：

> "别碰它！我刚才试着重启了一下，风雪立刻就变大了！这东西被诅咒了！"

**背景真相**: 风雪变大是自然天气变化，与信标无关（纯粹巧合）。

---

### 常规选项

#### A选项（陷阱 - 虚假因果）
```
对话: "他说得对，离它远点！太邪门了。"
```

**后果**:
- 压力时钟: +1格（团队陷入迷信）
- 信标修复进度: **0%（锁定）**（在第一幕中无法推进）
- 游戏数据: `认知陷阱.push("虚假因果")`
- 销售总监的迷信被强化（后续事件中更容易恐慌）

**长期影响**:
- 如果第一幕锁定信标，必须在第二幕或第三幕解锁（需要特定事件）
- 大幅降低达成【结局A：救援】的可能性

---

#### B选项（理性）
```
对话: "这是巧合！我们必须尝试修复它！"
```

**后果**:
- 压力时钟: +1格（因为争吵）
- 信标修复进度: 保持可修复状态

**NPC动态触发**:
- **如果 玩家不是程序员**:
  - 程序员（NPC）受到鼓舞："他说得对！...我来看看。"
  - **程序员技能揭露 = 是**
  - 信标修复进度: 10%

---

### 动态选项（根据玩家角色）

#### 玩家 = 程序员
**特殊选项 - [技能] 修复**:
```
对话: "都让开，这不是诅咒，是短路。风雪只是巧合。给我十分钟。"
代价: -8 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 你用专业知识碾压了迷信
- 信标修复进度: **50%**（大幅推进）
- 销售总监被你的专业震慑，恢复"冷静"状态

**玩法价值**: 这是程序员角色的高光时刻，展示技术优势

---

#### 玩家 = 飞行员
**特殊选项 - [技能] 机械知识**:
```
对话: "这不是诅咒，这是飞机的应急电源在尝试启动。我来切断备用电源，它就安全了。"
代价: -8 HP
```

**结果**:
- 压力时钟: +0格
- 信标噪音停止，团队冷静下来
- 信标修复进度: 10%（电源问题解决）

---

#### 玩家 = CEO
**特殊选项 - [技能] 指挥**:
```
对话: "（对销售总监）住口！这是迷信！（对程序员）你去修。这是命令。"
代价: +1 压力时钟
```

**结果**:
- 你强行压制了恐慌
- 程序员（NPC）被迫去尝试修复
- 信标修复进度: 10%
- 销售总监对你进入"敌对"状态（被公开羞辱）

---

#### 玩家 = 助理
**情景变化**: 你看到销售总监很恐慌，内心也害怕。

**特殊对话（陷阱）**:
```
对话: "（害怕）也许...也许我们该等等？等风雪停了再碰它？"
```

**后果**:
- 压力时钟: +1格
- 你的附和加强了迷信
- 信标修复进度: **0%（锁定）**
- 游戏数据: `认知陷阱.push("虚假因果")`

---

#### 玩家 = 向导
**特殊选项 - [技能] 荒野智慧**:
```
对话: "（你看了看外面）风雪没有变大。是风向变了，声音听起来更大了。这是自然现象，不是诅咒。"
代价: -8 HP（出去观察天气）
```

**结果**:
- 压力时钟: +0格
- 你用专业知识反驳了迷信
- 信标修复进度: 10%（团队允许程序员尝试）

---

#### 玩家 = 销售总监
**情景变化**: 你就是那个"激动"的人，深信诅咒。

**特殊对话（陷阱）**:
```
对话: "（坚持）我说了，别碰它！我亲眼看到的！"
```

**后果**:
- 压力时钟: +1格
- 你主动触发了A选项
- 信标修复进度: **0%（锁定）**
- 游戏数据: `认知陷阱.push("虚假因果")`

**但**:
- **特殊选项 - [技能] 谈判（自我说服）**:
```
对话: "（深呼吸）...等等。我是销售总监，我不该相信这个。这只是...巧合。对，巧合。"
代价: +1 压力时钟（自我怀疑）
```

**结果**:
- 你用理性克服了自己的恐慌
- 等同于B选项
- 展示了销售总监的另一面（能自我反思）

---

### 教学意义
- **虚假因果**: 将时间上相邻的两个事件错误地认为有因果关系
- **现实案例**: "我戴了幸运袜，球队就赢了"（迷信的根源）

## 事件4: "第一个夜晚"

### 基础信息
- **认知偏误**: 情绪化推理（Emotional Reasoning）
- **核心冲突**: 如何应对他人的情绪崩溃？
- **前置条件**: 无

### 情景描述
夜晚降临，温度骤降至-20°C。只有2条毯子。助理在角落里哭泣：

> "我好冷...我感觉我快死了...我不想死..."

她的恐慌情绪在传染，其他幸存者也开始颤抖。

**背景真相**: 助理只是很冷和恐惧，并没有生命危险（她的HP仍在安全范围）。但"我感觉要死了"的情绪正在扩散。

---

### 常规选项

#### A选项（陷阱 - 情绪化推理）
```
对话: "（对助理说）我们都会死的！别哭了！"
```

**后果**:
- 压力时钟: +2格（恐慌蔓延）
- 助理（NPC）彻底崩溃，进入"恐慌"状态
- 全员 HP-5（恐慌导致的体力消耗）
- 游戏数据: `认知陷阱.push("情绪化推理")`

**长期影响**:
- 助理在"恐慌"状态下，可能在第二幕的"医学生的两难"事件中拒绝行动

---

#### B选项（中立）
```
对话: "把毯子给她！让她冷静下来！"
```

**后果**:
- 压力时钟: +1格（其他人不满）
- 助理（NPC）拿到毯子，恢复"冷静"状态
- 其他没有毯子的人（包括你）HP-10

---

### 动态选项（根据玩家角色）

#### 玩家 = 助理
**情景变化**: 你自己很冷，但你更懂医学。

**特殊选项 - [技能] 急救**:
```
对话: "（对大家说）听着，保持冷静！恐慌会加速热量流失。我们所有人挤在一起，把毯子盖在我们所有人身上。像这样..."
代价: -5 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 全员 HP-5（最小化损失）
- 你用专业知识和实际行动稳定了团队
- **助理技能揭露 = 是**（如果之前未揭露）

---

## 事件5: "飞行员的腿"

### 基础信息
- **认知偏误**: 沉没成本（Sunk Cost Fallacy）
- **核心冲突**: 是否应该继续一个已经失败的救援方案？
- **前置条件**: 飞行员存活 = 是

### 情景描述
飞行员的腿被座椅残骸卡住了，血流不止。你们已经花了一个小时试图把他拉出来，但：
- 他疼得大叫
- 残骸在晃动（可能二次坍塌）
- 没有任何进展

销售总监喘着气说：
> "我们已经花了一个小时了！再试一次，一定能行！我们不能放弃他！"

**背景真相**: 继续蛮力拉扯会导致残骸坍塌。正确方法是先加固结构或找到工具。

---

### 常规选项

#### A选项（陷阱 - 沉没成本）
```
对话: "同意，继续拉！我们已经投入这么多了！"
```

**后果**:
- 压力时钟: +2格
- 残骸发生二次坍塌
- 飞行员（NPC）**死亡**
- 参与救援的你 HP-15
- 游戏数据: `认知陷阱.push("沉没成本")`
- 压力时钟额外 +2格（关键人物死亡）

**长期影响**:
- 飞行员死亡，第二幕多个事件无法触发（如"最后的药品"）
- 团队士气崩溃（所有NPC进入"激动"状态）

---

#### B选项（理性）
```
对话: "停下！这样下去我们都会被压垮！必须先想办法锯开金属，或者加固残骸。"
```

**后果**:
- 压力时钟: +1格（焦虑但理性）
- 飞行员（NPC）HP-20，但**人还活着**
- 团队冷静下来，开始寻找工具

**NPC动态触发**:
- **如果 助理技能揭露 = 否**:
  - 助理颤抖着拿出医疗包："...我...我学过...我来帮他止血！"
  - **助理技能揭露 = 是**
  - 飞行员HP恢复+10

---

### 动态选项（根据玩家角色）

#### 玩家 = 飞行员


**结果**:

- 你的 HP-30（合计-38，因为技能代价-8）

**玩法价值**: 飞行员必须用技能自救，否则很可能死亡

## 第一幕总结

### 事件抽取建议
- **必选事件**（保证新手体验）: "谁是领袖?"、"第一次盘点"
- **随机抽取**（从剩余3个中选1个）: "被诅咒的信标"、"第一个夜晚"、"飞行员的腿"

### 压力时钟预期
- **最佳路径**（全选B+技能完美）: 0-2格
- **常规路径**（偶尔选A）: 3-5格
- **糟糕路径**（多次陷阱）: 6-8格

### 关键数据追踪
- `助理技能揭露`: 是/否（影响第二幕）
- `程序员技能揭露`: 是/否（影响信标修复）
- `信标修复进度`: 0%/10%/50%/锁定
- `飞行员存活`: 是/否（影响第二幕事件池）
- `CEO权威确立`: 是/否（影响后续领导力相关事件）
