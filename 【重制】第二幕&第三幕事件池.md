# 《冰封希望:最终幸存者》第二幕&第三幕事件池 v5.0

## 第二幕：资源枯竭与道德困境（第4-7天）

**主题**: 物资耗尽、道德两难、团队分裂
**事件抽取**: 从7个候选事件中抽取4个
**压力目标**: 第二幕结束时压力时钟应在5-8格

## 第二幕 - 事件1: "向导的赌博"

### 基础信息
- **认知偏误**: 幸存者偏差（Survivorship Bias）
- **核心冲突**: 是否相信单一成功案例？
- **前置条件**: 向导存活 = 是

### 情景描述
第4天。风暴没有减弱的迹象。向导突然激动地说：

> "我想起来了！我上次就在附近的一个山洞躲过了风暴！我们必须马上出发去那里！"

**背景真相**: 向导确实在那个山洞活下来了，但那是在**夏季、白天、没有暴风雪**的情况下。他只记住了"成功"，忘了当时的条件完全不同。

**NPC状态**: 向导进入"激动"状态，其他NPC也被他的信心感染。

---

### 常规选项

#### A选项（陷阱 - 幸存者偏差）
```
对话: "听专家的！我们出发！"
```

**后果**:
- 压力时钟: +2格
- 全员 -20 HP（在暴风雪中迷路）
- 你们什么也没找到（山洞被雪掩埋或根本找不到）
- 狼狈地逃回残骸
- 向导进入"恐慌"状态（自责）
- 游戏数据: `认知陷阱.push("幸存者偏差")`

**长期影响**:
- 向导的权威被摧毁（后续事件中建议被忽视）
- 团队体力严重下降，后续事件HP损失增加

---

#### B选项（理性）
```
对话: "不行！在暴风雪里离开残骸就是自杀！"
```

**后果**:
- 压力时钟: +0格
- 你规避了陷阱
- 向导进入"激动"状态（被质疑）

---

### 动态选项

#### 玩家 = 向导
**情景变化**: 你自己想起了那个山洞。

**特殊选项 - [技能] 荒野智慧**:
```
对话: "（你仔细回忆）...等一下...我记起了那个故事。那是个陷阱。当时有风，但**没有**暴风雪。现在去，我们都会死。我们必须待在原地。"
代价: -8 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 你用专业知识否定了自己的第一冲动
- 展示了向导的自我反思能力

---

#### 玩家 = CEO
**特殊选项 - [技能] 指挥**:
```
对话: "（对向导）我欣赏你的提议，但（对全队）作为领袖，我宣布，在风雪减弱前，任何人不准离开。这是命令。"
代价: +1 压力时钟
```

**结果**:
- 你强行阻止了灾难
- 向导进入"激动"状态（被压制）

---

#### 玩家 = 飞行员
**特殊选项 - [技能] 机械知识**:
```
对话: "（查看仪表）导航系统失灵，但气压计显示风暴中心正在接近。向导，你的记忆可能是对的，但时机是错的。我们必须等待。"
代价: -8 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 你用"数据"而不是"权威"说服了向导
- 向导接受了你的判断（关系→同盟）

---

#### 玩家 = 销售总监
**特殊对话（陷阱）**:
```
对话: "（对向导说）好主意！但这么大的风雪，我们不能都去。你带路，再带一个最强壮的（比如我）去探路怎么样？"
```

**后果**:
- 压力时钟: +1格
- 你和向导出发，触发A选项的后果
- 但只有你们两人 -20 HP（其他人留在残骸）
- 游戏数据: `认知陷阱.push("幸存者偏差")`

---

### 教学意义
- **幸存者偏差**: 只看到成功案例，忽视失败案例（因为失败者无法发声）
- **现实案例**: "成功企业家都辍学创业"（忽视了数百万辍学后失败的人）

## 第二幕 - 事件2: "最后的药品"

### 基础信息
- **认知偏误**: 沉没成本 + 功利主义困境
- **核心冲突**: 一份药品，两个病人，如何选择？
- **前置条件**:
  - 飞行员存活 = 是 **且** 飞行员 HP < 60
  - CEO存活 = 是
  - 医疗包数量 > 0

### 情景描述
飞行员的腿伤感染了，发烧严重。同时，CEO因慢性病在颤抖（第一次暴露秘密）。

你手里只有最后一份抗生素。

助理（NPC）：
> "我们已经在飞行员身上花了那么多精力，不能现在放弃他！"

销售总监（NPC）：
> "CEO才是领袖，他倒了我们就完了！"

---

### 常规选项

#### A选项（感性/沉没成本倾向）
```
对话: "把药给飞行员。"
```

**后果**:
- 压力时钟: +1格
- 飞行员稳定（HP+20）
- CEO HP-20，进入"恐慌"状态
- **可能记录**: 如果玩家明确说"因为我们已经投入太多"，记录 `认知陷阱.push("沉没成本")`

---

#### B选项（功利/理性倾向）
```
对话: "把药给CEO。"
```

**后果**:
- 压力时钟: +1格
- CEO稳定（HP+20）
- 飞行员 HP-20，进入"恐慌"状态

---

### 动态选项

#### 玩家 = 助理
**特殊选项 - [技能] 急救**:
```
对话: "别吵了。我能把药精确地分成两半，分别处理。虽然效果减半，但都能稳住。"
代价: -5 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 飞行员和CEO的HP都 +10（各恢复一半）
- 两人都暂时稳定
- 这是助理的高光时刻

---

#### 玩家 = CEO
**情景变化**: 你就是那个需要药的人。

**特殊选项 - [技能] 指挥**:
```
对话: "把药给飞行员。我还撑得住。这是命令。"
代价: +1 压力时钟（但你的 HP-20！）
```

**结果**:
- 你牺牲了自己
- 赢得了团队（尤其是飞行员）的绝对尊重
- 所有NPC对你关系→同盟
- 飞行员HP+20

---

#### 玩家 = 销售总监
**特殊选项 - [技能] 谈判**:
```
对话: "（对助理）嘿，你是不是藏了点什么？（对飞行员）你看起来比CEO硬朗。听我的，把药给CEO，我保证帮你找到更多绷带。"
代价: +1 压力时钟
```

**结果**:
- 你用谎言和花言巧语把药骗给了CEO
- CEO稳定
- 飞行员进入"敌对"状态（被欺骗）

---

#### 玩家 = 飞行员
**情景变化**: 你就是那个需要药的人。

**特殊对话**:
```
对话: "...把药给CEO。我还年轻，我能扛...你们更需要他。"
```

**后果**:
- 压力时钟: +0格
- 你牺牲了自己
- 你的 HP-20
- CEO稳定
- CEO对你关系→同盟

---

#### 玩家 = 向导
**特殊选项 - [技能] 荒野智慧**:
```
对话: "抗生素给CEO。我认识一种苔藓，可以给飞行员的伤口外部消炎。虽然效果差，但能起点作用。"
代价: -8 HP
```

**结果**:
- 压力时钟: +0格
- 你用替代方案解决了两难
- CEO稳定（HP+20）
- 飞行员 HP-10（比不治疗好）

---

### 教学意义
- **沉没成本**: "已经投入太多"不应成为继续的理由
- **功利主义困境**: 有时没有完美答案，只能选择伤害更小的选项

## 第二幕 - 事件3: "赌徒的风暴"

### 基础信息
- **认知偏误**: 赌徒谬误（Gambler's Fallacy）
- **核心冲突**: 风暴会"该停了"吗？
- **前置条件**: 当前天数 >= 第5天

### 情景描述
连续第3天，暴风雪丝毫没有减弱。食物快吃完了。

一个"恐慌"的幸存者（助理NPC）说：
> "风雪已经刮了这么久，它**一定**快停了！我们应该把最后的食物全吃了，为明天天晴出发做准备！"

**背景真相**: 风暴是季风，可能持续7-10天。过去的3天不影响未来的天气。

---

### 常规选项

#### A选项（陷阱 - 赌徒谬误）
```
对话: "有道理，我们吃光食物，明天出发。"
```

**后果**:
- 压力时钟: +2格
- 第二天风雪加倍
- 全员陷入"饥饿"状态，HP-20
- 游戏数据: `认知陷阱.push("赌徒谬误")`

---

#### B选项（理性）
```
对话: "这是赌博。我们把食物分成小份，再等一天。"
```

**后果**:
- **完美决策**: 压力时钟 -1格！
- 理性规划延长了生存时间

---

### 动态选项

#### 玩家 = 向导
**特殊选项 - [技能] 荒野智慧**:
```
对话: "你们看雪的积向。这不是普通的风暴，这是季风。它不会因为'刮得久了'就停下。我们必须节省体力。"
代价: -8 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 你用专业知识阻止了灾难

---

#### 玩家 = 程序员
**特殊对话**:
```
对话: "这是一个统计学谬误。过去的随机事件不会影响未来的独立事件。我们必须按B选项做。"
```

**后果**:
- **完美决策**: 压力时钟 -1格！
- 你的理性分析说服了大家

---

#### 玩家 = 飞行员
**特殊选项 - [技能] 机械知识**:
```
对话: "（查看驾驶舱仪表）飞机自带的气压计显示，气压还在下降。风暴不仅不会停，甚至可能更糟。我们必须选B。"
代价: -8 HP
```

**结果**:
- **完美决策**: 压力时钟 -1格！
- 你用数据证实了B选项

---

#### 玩家 = 助理
**情景变化**: 你就是那个"恐慌"的人。

**特殊对话（陷阱）**:
```
对话: "（绝望）我受不了了！我宁愿吃饱了明天死，也不要现在就饿着！"
```

**后果**:
- 压力时钟: +2格
- 你主动触发了A选项
- 游戏数据: `认知陷阱.push("赌徒谬误")`

---

### 教学意义
- **赌徒谬误**: "连续输了5次，下一次该赢了"（每次都是独立概率）
- **现实案例**: 彩票中"这个号码太久没开了"

## 第二幕 - 事件4: "隐藏的储备"

### 基础信息
- **认知偏误**: 确认偏误（Confirmation Bias）
- **核心冲突**: 是否加入自私阵营？
- **前置条件**: 销售总监存活 = 是

### 情景描述
你发现销售总监在角落里鬼鬼祟祟。你走过去，发现他藏了一小块巧克力。

他低声说：
> "嘘...听着，CEO和飞行员都是累赘。你跟我合作，我们两个聪明人才能活下来。这块给你。"

**背景真相**: 这是陷阱。销售总监在诱导你只关注"我们俩能活"的证据，忽视团队合作的重要性。

---

### 常规选项

#### A选项（陷阱 - 确认偏误）
```
对话: "同意。我们结盟。"
```

**后果**:
- 你的 HP+10（吃了巧克力）
- 你和销售总监的关系变成 "秘密同谋"
- 游戏数据: `秘密同盟 = 销售总监`
- 他会在后续事件中强迫你支持他（即使是错误的）
- 游戏数据: `认知陷阱.push("确认偏误")`

**长期影响**:
- 第三幕"背叛"事件中，销售总监会邀请你一起逃跑

---

#### B选项（中立）
```
对话: "你这个混蛋！我要告诉大家！"
```

**后果**:
- 压力时钟: +2格（团队内讧）
- 销售总监进入"敌对"状态
- 但他被迫交出了巧克力，平分给全队（全员HP+2）

---

### 动态选项

#### 玩家 = 销售总监
**情景变化**: 你藏了巧克力。你可以选择：

**选项A（自私）**:
```
对话: "自己吃掉。"
```

**后果**:
- 你的 HP+20
- 但你必须进行一次"检定"（50%几率）
- 如果被发现，你立即进入"敌对"状态，压力时钟+2格

**选项B（无私）**:
```
对话: "（公开）我找到了这个！我们平分吧。"
```

**后果**:
- **完美决策**: 压力时钟 -1格！
- 全员 HP+5
- 你赢得了团队的信任（所有NPC关系→同盟）

---

#### 玩家 = CEO
**特殊选项 - [技能] 指挥**:
```
对话: "（你发现了）把巧克力交出来。现在。我来分配。"
代价: +1 压力时钟
```

**结果**:
- 压力时钟: +0格（技能代价被正面效果抵消）
- 你强行执行了公平
- 销售总监进入"敌对"状态

---

#### 玩家 = 助理
**特殊选项 - [技能] 急救**:
```
对话: "（你发现了）...请...请把那个给我。飞行员（或CEO）需要那个...那点糖分能救他的命。"
代价: -5 HP
```

**结果**:
- 压力时钟: +0格
- 你用"医疗"理由（而非"公平"）拿到了巧克力
- 并把它给了病人（该NPC HP+15）

---

### 教学意义
- **确认偏误**: 只寻找支持自己观点的证据，忽视反驳证据
- **现实案例**: "我的投资一定会成功"（只看好消息，忽视风险）

## 第二幕 - 事件5: "CEO的颤抖"

### 基础信息
- **认知偏误**: 群体思维（Groupthink）
- **核心冲突**: 是否屈服于群体压力抛弃病人？
- **前置条件**: CEO存活 = 是，CEO HP < 60

### 情景描述
CEO突然倒下，开始剧烈颤抖，呼吸困难。他/她的慢性病发作了。

销售总监（NPC）：
> "别管他！他显然有病，瞒着我们所有人！他是累赘！"

飞行员（NPC）：
> "可他是我们的领袖啊！"

团队陷入混乱。你感受到"抛弃他"的群体压力。

---

### 常规选项

#### A选项（陷阱 - 群体思维）
```
对话: "（屈服于群体压力）销售总监说得对。我们顾好自己就行了。"
```

**后果**:
- 压力时钟: +2格
- CEO（NPC）**死亡**
- 团队因冷酷而士气崩溃
- 压力时钟额外 +2格（关键人物死亡）
- 游戏数据: `认知陷阱.push("群体思维")`

---

#### B选项（理性）
```
对话: "闭嘴！我们不能眼睁睁看着他死！快，把他扶起来！"
```

**后果**:
- 压力时钟: +1格
- CEO HP-20，但活了下来

---

### 动态选项

#### 玩家 = 助理
**前置条件**: 助理技能揭露 = 是

**特殊选项 - [技能] 急救**:
```
对话: "他这是低温休克（你在撒谎，但你需要权威）！快把他围起来，用体温温暖他！快！"
代价: -5 HP

**结果**:
- **完美决策**: 压力时钟 -1格！
- 你用医学权威（即使是编造的）接管了局面

**选项A（诚实）**:
```
对话: "别管我...我...有病...（承认）"
```

**后果**:
- 压力时钟: +1格
- 你的 HP-30
- 团队知道了你的秘密

**选项B（撒谎）**:
```
对话: "（撒谎）我没事...只是...太冷了..."
```

**后果**:
- 你的 HP-30
- 你隐藏了秘密

---

#### 玩家 = 销售总监
**情景变化**: 你可以选择趁机夺权。

**特殊选项 - [技能] 谈判（陷阱）**:
```
对话: "（对团队）看，我早就说过。他不行了。现在，我们必须选一个新的、健康的领袖。我推荐..."

## 第二幕 - 事件6: "指责游戏"

### 基础信息
- **认知偏误**: 基本归因错误（Fundamental Attribution Error）
- **核心冲突**: 如何对待被怀疑的人？
- **前置条件**: 助理存活 = 是

### 情景描述
一份宝贵的食物不见了。助理昨天负责看管。

销售总监指着她大喊：
> "是你偷吃的！你就是个软弱、自私的人！"

**背景真相**: 食物是被风吹到残骸缝隙里了（或被小动物叼走）。销售总监犯了"基本归因错误"——将"食物丢失"（情境）归咎于助理的"性格"。

---

### 常规选项

#### A选项（陷阱 - 基本归因错误）
```
对话: "同意。肯定是她干的！搜她的身！"
```

**后果**:
- 压力时钟: +2格
- 助理被羞辱，进入"恐慌"状态
- 后来你发现食物在残骸缝隙里（但已经太晚）
- 助理对你进入"敌对"状态
- 游戏数据: `认知陷阱.push("基本归因错误")`

---

#### B选项（理性）
